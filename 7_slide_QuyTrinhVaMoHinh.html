IQR = Q3 - Q1 | Khoảng: [Q1-1.5*IQR, Q3+1.5*IQR]
3. SMOTE (Xử lý mất cân bằng)
Tạo mẫu tổng hợp lớp thiểu số nhằm cân bằng tập huấn luyện.
4. Tối ưu hóa (GridSearchCV)
Kết hợp 5-fold Cross-validation để tìm tham số tối ưu.

2.4 Thuật toán đề xuất

1. Logistic Regression (LR)

Mô hình nền tảng, dễ diễn giải quan hệ tuyến tính.
P(Y=1|X) = 1 / (1 + e-(β₀ + ΣβᵢXᵢ))

2. Random Forest (RF)

Ensemble Learning (Bagging). Giảm overfitting, xử lý tốt quan hệ phi tuyến phức tạp.

3. XGBoost (Extreme Gradient Boosting)

Thuật toán Boosting hiệu suất cao, tối ưu hóa mạnh mẽ thông qua Regularization.
